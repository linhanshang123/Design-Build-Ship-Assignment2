## Project
Build Assignment 2 as a Day Planner web app.

## Current stage
We are currently in planning + scaffolding.

## Goal
Create a lightweight multi-page planner that helps a user manage tasks and notes by day.

## Tech
- Next.js
- Tailwind CSS
- App Router

## Required routes
- `/` home page
- `/new` add task form
- `/day/[date]` dynamic day page
- `/week` week overview

## Current priorities
1. Scaffold the Next.js app
2. Set up Tailwind
3. Create the 4 required routes
4. Add a shared navigation bar
5. Use simple placeholder content first

## Constraints
- Keep it simple
- Use client-side state only
- No database
- No auth
- No external APIs

## Style
Clean, calm, simple, polished.

## Working style
- Work in small steps
- Build the route structure first
- Do not over-engineer
- Suggest commits, but do not auto-commit unless asked
